## Powerball Pattern Intelligence

⚠️ Disclaimer:
Powerball draws are random.
This project does NOT predict future numbers.

### What this does:
- Pattern classification
- Historical similarity analysis
- Draw clustering
- Strategy comparison

### ML Models:
- Random Forest (pattern classifier)
- Gradient Boosting (pattern classifier)
- KNN (similarity engine)
- KMeans (draw clustering)

### Deployment:
- Hosted on Render
- Models trained via background worker
